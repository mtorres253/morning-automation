# Job Search Assets

This directory stores your professional materials for the job-search skill.

## Files

- **resumes.md** — Your resume(s) in Markdown format. Update this with your latest experience, education, and skills. The job-search skill references this when tailoring applications.

- **cover_letters.md** — Cover letter templates in Markdown format. Store your template(s) and variations here. The skill can customize these for specific roles and companies.

## How They're Used

When the job-search skill finds a promising opportunity, it can:
1. Read your resumes to extract relevant experience
2. Customize a cover letter template for the specific role/company
3. Include tailored versions in your digest or directly apply if configured

## Format Tips

Keep these files clean and structured:
- Use clear section headers (## Experience, ## Skills, etc.)
- Use consistent formatting so parsing is easy
- Include multiple resume versions if you have them (e.g., technical PM vs. general PM)
- Maintain 2-3 cover letter templates for different role types
