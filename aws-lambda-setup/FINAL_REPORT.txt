IMPORTANT NOTES
================================================================================

1. CREDENTIALS
   • Gmail OAuth token is passed as environment variable (encrypted by AWS)
   • AWS IAM credentials used for deployment only (not stored in Lambda)
   • Store credentials safely - don't commit to git

2. SECURITY
   • Lambda functions have minimum necessary permissions (principle of least privilege)
   • S3 bucket is private (no public access)
   • Logs retained for 7 days then deleted
   • No internet access required (uses AWS service endpoints)

3. SES EMAIL
   • If in sandbox mode, emails only go to verified addresses
   • Request production access in AWS console (usually instant)
   • Sender email must be verified in SES

4. TESTING
   • Test functions before going live
   • Check CloudWatch logs for errors
   • Verify emails arrive in inbox
   • Monitor for first 24 hours

5. CUSTOMIZATION
   • Edit lambda_functions/*.py to change behavior

   • Run python3 deploy.py again to update
